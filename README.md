Today's Weather Forecast

Description
For Phase-1 Final Project for Flatiron School. For this project, we were asked to creat a single-page application using only HTML, CSS, and JavaScript as well as incorporate a public API which would provide the data for our web page.

I decided to go with the OpenWeatherMap API for this project because it is free, public, and relatively simple.

"Today's Weather Forecast" is a single-page application that provides the user with real-time weather information as well as a detailed 3-hourly forecast for any city across the globe. This page fetches accurate and up-to-date weather info, including temperature, weather conditions, fun icons, and more. The web page is designed to be interactive, simple, and visually appealing.

Features
~ Displayes current weather information for any given city.
~ Provides a 3-hourly forecast for the next 24 hours.
~ Interactive and responsive interface with dynamic updates.
~ Real-time data fetching without page reloads.

Installation and Usage Instructions

Installation
1.) Clone the repository to your local machine:
git clone https://github.com/your-username/your-repository-name.git

2.) Navigate to the project directory:
cd your-repository-name

3.) Open the index.html file in your browser to run the application.

Usage
1.) Enter the name of a city in the input field.
2.) Click the Search button or press the Enter key to fetch the weather data.
3.) View the current weather details and the 3-hourly forecast.
4.) Hover over the weather icon for additional information.
